Инструкция для импорта данных через Postman в Altcraft Platform.

Инструкция

Другими словами необходимо при помощи Postman в базе данных Altcraft Platform создать запись о профиле, а именно фамилию, имя, email, дату рождения. Эти данные можно передать в теле метода POST.


Видео инструкция


Пошаговая инструкция

  • Открыть Postman или его web-версию.
  • Выбрать “Workspaces”.
  • Создать коллекцию, нажав на "Collections".
  • Нажать на кнопку "+" или "Create Collection".
  • Дать название осмысленное название коллекции, например, "Работа с профилем".
  • Создать запрос, нажав на "Add a request".
  • Дать название новому запросу согласно цели запроса, например, "Импорт одного профиля".
  • Изменить тип запроса с GET на POST, именно в теле метода POST будут введены данные для отправки в БД.

  • В поле “Enter request URL” вставить ссылку API платформы https://ru.altkraft.com/api/v1.1/profiles/import .
  • Выбрать пункт “Body” для настройки деталей запроса.
  • Нажать радиокнопку “raw” для выбора типа форматирования - Postman отправляет запрос с таким типом без проверки.
  • Выбрать формат отправляемых данных - JSON.

  • В теле запроса указать необходимые данные о профиле(комментарии удалить):
    {"token": "421dcf213c0b4089a839b08333de1f5f", // - API токен (используются для авторизации в аккаунте Altcraft платформы при использовании API)
    "db_id": 2, // Идентификатор базы данных
    "matching": "email", // алгоритм по которому в дальнейшем можно будет найти создаваемый профиль и изменить его
    "email": "marta0706@example.com", // должен быть уникальным во избежании ошибок
    "data":
    {
    "_fname": "John",
    "_lname": "Doe",
    "_bdate": "2003-01-19T21:00:00Z" //- дату вписать в соответствии со стандартом
    }}
  • Нажать на кнопку “Send” для отправки запроса.

 

Результат:"Запрос успешно выполнен". Статус код - "200".

При необходимости сохранить запрос:

  • Нажать кнопку "Save".
  • По умолчанию запрос сохранится в текущую коллекцию. 

Посмотр созданного профиля

  • Создать Post запрос.
  • В URL указать “https://ru.altkraft.com/api/v1.1/profiles/get”
  • В теле запроса указать Идентификатор базы данных, email, API токен:
    {
    "db_id": 2, //
    "email": "marta0706@example.com",
    "token": "421dcf213c0b4089a839b08333de1f5f"
    }
  • Нажать на кнопку "Send".

Ответом будет информация о профиле.

 

Для сохранения запроса:

  • Hажать кнопку "Save".
  • Выбрать текущую коллекцию.
  • Hажать кнопку "Save".
