Инструкция для импорта данных через Postman в Altcraft Platform.

Другими словами необходимо при помощи Postman в базе данных Altcraft Platform создать запись о профиле, а именно ФИ,email, дату рождения. Эти данные можно передать в теле метода POST.


Для просмотра видео инструкции нажмите на ссылку -


Пошаговая инструкция.

  1. Открыть Postman или его web версию.
  2. Выбрать “Workspaces”.
  3. Изменить тип запроса с GET на POST, именно в теле метода POST будут введены данные для отправки в БД.
  4. В поле “Enter request URL” вставить ссылку API платформы https://ru.altkraft.com/api/v1.1/profiles/import .
  5. Выбрать пункт “Body” для настройки деталей запроса.
  6. Нажать радиокнопку “raw” для выбора типа форматирования - Postman отправляет запрос с таким типом без проверки.
  7. Выбрать формат отправляемых данных - JSON.
  8. В теле запроса указать необходимые данные о профиле:
    {"token": "421dcf213c0b4089a839b08333de1f5f", // - API токен (используются для авторизации в аккаунте Altcraft платформы при использовании API)
    "db_id": 2, // Идентификатор базы данных
    "matching": "email", // алгоритм по которому в дальнейшем можно будет найти создаваемый профиль и изменить его
    "email": "marta0706@example.com", // должен быть уникальным во избежании ошибок
    "data": {
    "_fname": "John",
    "_lname": "Doe",
    "_bdate": "2003-01-19T21:00:00Z" - дату вписать в соответствии со стандартом
    }
    }
  9. Нажать на кнопку “Send” для отправки запроса.
  10. Перед вами результат запроса и статус код.
    11. При необходимости сохранить запрос:
    1. Нажать кнопку "Save".
    2. Дать название запросу, например, “Импорт одного профиля”.
    3. Выбрать коллекцию для сохранения или создать новую.

    Посмотреть созданный профиль можно следующим образом:

    1. Создать Post запрос.
    2. В URL указать “https://ru.altkraft.com/api/v1.1/profiles/get”
    3. В теле запроса указать Идентификатор базы данных, email, API токен:
      {
      "db_id": 2, //
      "email": "marta0706@example.com",
      "token": "421dcf213c0b4089a839b08333de1f5f"
      }
    4. Нажать на кнопку "Send".
      Ответом будет информация о профиле.
